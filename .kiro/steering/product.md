# Product Overview

InsScan is a shipment tracking platform for PPJK (customs broker) and freight forwarder staff. The core value proposition is automating manual data entry from Bill of Lading documents.

## Current Features

- HS Code lookup and tariff information retrieval from INSW API
- Excel-based data import/export for customs documentation
- Automatic ETA fetching and updates (in progress)

## Target Users

- Operational staff at PPJK companies who manually input shipment data
- Freight forwarder teams managing multiple shipments

## Key Problems Solved

- Reduces manual data entry time (from 15 minutes to 2-3 minutes per shipment)
- Minimizes human error in customs documentation
- Centralizes shipment information that's typically scattered across email/WhatsApp/Excel

## Product Evolution

The project is pivoting toward a browser extension model (BL Parser Extension) that will:
- Parse Bill of Lading PDFs automatically
- Auto-fill forms via keyboard shortcuts
- Work with CEISA and internal PPJK software
- Use credit-based monetization (1 credit = 1 parsing)
